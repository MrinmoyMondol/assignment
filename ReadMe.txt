I try to create a simple contact management system. Here is the feature list of task-

1. Authentication (Login & Registration)

2. New Contact entry

3. Edit Contact

4. Update Contact



For Running this project -

1/ Need the latest PHP version Xampp

https://www.apachefriends.org/download.html

2/ After set up the environment -
	
	a) THe project file needs to placed in this location ( htdocs folder)
		C:\xampp\htdocs  
	b) open XAMPP start - apache & mysql
	
	c) In this directory using cmd write this line-
		" php artisan serve "
	
	d) Get this and  placed in browser addressbar-  http://127.0.0.1:8000/
